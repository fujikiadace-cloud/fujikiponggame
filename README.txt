# 通信対戦（サーバーURL不要の1URL版）

## これが一番カンタンです
このサーバーは **Webページ配信（WEBサーバー）** と **通信（WebSocket）** を同じURLでまとめています。

### 起動
1) Node.js 18+ 推奨
2) `cd server`
3) `npm i`
4) `npm start`

### 開き方（2台のスマホ）
同じWi‑Fi内なら、スマホのブラウザで
`http://<PCのIPアドレス>:8787/`
を開く（2台とも同じURL）

※PCのIP例：192.168.0.10 など（Windows: ipconfig / Mac: ifconfig）

### 対戦
1) 片方が「部屋を作る」→コードが出る
2) もう片方がコードを入力して「入る」
3) 「ゲーム開始」

## 注意（iPhoneでインターネット越し）
外部公開してiPhoneで使う場合は HTTPS が必要になることが多いです。
その場合は https + wss 対応（リバースプロキシ等）が必要です。
